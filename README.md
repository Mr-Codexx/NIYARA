## 🚀 Overview

**NIYARA** is a modern, high-performance brand showcase website for a premium hygiene and tissue product company. The platform is meticulously crafted to reflect the brand's core values of sustainability, quality, and innovation. It serves as a digital storefront to elegantly display product categories, communicate the brand's deep commitment to eco-friendly practices, and provide seamless connection points for distributors and customers.

🔗 **Live Experience:** [https://niyara.vercel.app/](https://niyara.vercel.app/)

## 🛠 Technology Stack

*   **Framework:** [Next.js 14](https://nextjs.org/) (React) – for server-side rendering, static site generation, and optimal performance.
*   **Frontend Library:** [React 18](https://reactjs.org/) – for building a dynamic and component-based UI.
*   **Styling:** [Tailwind CSS](https://tailwindcss.com/) – for a utility-first, highly customizable, and responsive design.
*   **Deployment & Hosting:** [Vercel](https://vercel.com/) – for seamless Git integration, global edge network, and automatic SSL.

## 🚀 Getting Started

### Prerequisites
*   Node.js (v18 or later recommended)
*   npm or yarn

### Local Development
1.  **Clone the repository**
    ```bash
    git clone https://github.com/your-username/niyara.git
    cd niyara
    ```
2.  **Install dependencies**
    ```bash
    npm install
    # or
    yarn install
    ```
3.  **Run the development server**
    ```bash
    npm run dev
    # or
    yarn dev
    ```
4.  Open [http://localhost:3000](http://localhost:3000) to view it in the browser. The page will auto-update as you edit files.

### Production Build
To create an optimized production build:
```bash
